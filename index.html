渗透测试学习笔记（一）

发表于 2015-08-14   |  

虽然说之前稍微接触了渗透测试的小范围知识（也就是看了点《常用工具》里的泛泛介绍），现在才想到要正式记录自己关于这方面遇到的问题以及学习中的心得体会。希望自己能坚持下去。

测试环境

主机：Windows 10 ， Mac OS X 10.11

虚拟机：VMware Workstation 11

遇到的问题

1.在安装Kali Linux时候，提示网络中DHCP不存在（反应慢或者未联网）

解决方法： 应该在最初新建虚拟机导入ISO文件时，就在网络适配器中选择“桥接模式”。

2. 在安装Kali Linux时候，选择镜像网站后不能下载package

解决方法： 在最初选择语言和时区的时候，选择中国。目测美国镜像是被墙的。另外一定要确保以桥接模式联网。

3. 在安装Kali Linux之后，重启后在登陆界面闪现便是黑屏

解决方法：最终解决方法不是很科学，猜测是因为无法加载出图形界面而黑屏了。再三思索后用了官网的vmx文件，能够成功加载了。

4. 开机以后使用apt-get update命令，显示fetch … failed

解决方法：估计kali.org被墙，路由大法好。

5. 用户操作

更多详见：Linux增加／删除用户（组）

6. 尝试制作Portable Kali Linux

使用的是书中推荐的Rufus软件，很方便，制作也很快。但是在电脑BIOS设定以后，并没有从U盘启动，打算再次通过这个教程制作：How to make a bootable Kali Linux USB Flash Drive/Pendrive ［YouTube］

后来发现是启动方式问题，自己使用的是华硕电脑，BIOS设置中需要关闭secure boot, 启动CSM，最后重启时通过长按ESC键进入Boot Option。

7. 在USB闪存盘上安装系统以后，想要让系统保存修改的文件

详见：用Live U盘安装Kali Linux

8. VMware文件夹共享位置没有找到

解决方法：首先在VMware里面设置shared文件夹，优先级为最高。

You might be confused looking at the picture below, but the math is actually quite simple (albeit a bit elaborate). You’ll need the codes HERE, which will help you master this.

July simply has a code of 5.

20th is 6 because 7 goes into 20 twice, which is 14. 20 – 14 = 6.

2069 is 2 because the leap year code of 2068 is 1 and 2069 is 1 year after, so that’s 2.

The math can be difficult at first because there are a lot of codes, but it works out incredibly well once you pick it up. Here’s a separate example:

January 3, 2014 is a Friday, right?

So, January, according to the table, is 6, and we handle days by using multiples of 7. We don’t need to, so it’s actually just 3 in this case. Next we need to know the closest leap year, which was 2012. That has a year code of 1 and 2013 is 2 years after, so 3 again.

We get 6 + 3 + 3 = 12 – 7 = 5! FRIDAY!

添加SSH Key到Github

在本机设置SSH Key之后，需要添加到GitHub上，以完成SSH链接的设置。

1、打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件。此文件里面内容为刚才生成人密钥。如果看不到这个文件，你需要设置显示隐藏文件。 准确的复制这个文件的内容，才能保证设置的成功。

2、登陆github系统。点击右上角的 Account Settings—>SSH Public keys —> add another public keys

3、把你本地生成的密钥复制到里面（key文本框中）， 点击 add key 就ok了
