安装准备

  1. Node.js 在页面点击Install后下载安装即可

    ​ Mac OS 方法为brew install node

  2. Git

    1. 环境配置
    2. 也可以直接下载Git GUI省去环境配置这一步

      1. 可以在windows系统右键所需文件路径，点击Git Bash Here.
      2. 直接点击已经下载好的Git Bash.
      3. 或者利用下载Git GUI之后的Git Shell.

配置SSH

检查自己电脑上的SSH Key

提示：No such file or directory ，那么你是第一次使用Git，需要进行以下步骤。

​ 如果之前下载过Git GUI并安装，那么客户端已经帮你配置好Key，你可以直接跳到 使用Github Pages建立博客，如果之后在链接上出现问题，建议还是自己新建一个SSH Key。

生成新的SSH Key

会出现以下提示信息

回车即可（表示你的Key被保存在 /Users/your_user_directory/.ssh/id_rsa）

然后要求你输入密码：

再输入密码确认：

如果两次你都回车则表示输入为空。输入这个密码串是对你和Github之间的链接有了加密作用，防止别人远程登入修改你的项目。

添加SSH Key到Github

在本机设置SSH Key之后，需要添加到GitHub上，以完成SSH链接的设置。

1、打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件。此文件里面内容为刚才生成人密钥。如果看不到这个文件，你需要设置显示隐藏文件。 准确的复制这个文件的内容，才能保证设置的成功。

2、登陆github系统。点击右上角的 Account Settings—>SSH Public keys —> add another public keys

3、把你本地生成的密钥复制到里面（key文本框中）， 点击 add key 就ok了
